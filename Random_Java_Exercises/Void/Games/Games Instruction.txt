This is an example of **inheritance** in object-oriented programming (OOP), specifically using C\# or Java syntax (as they commonly use the `public void` structure).

The example shows a base class `Game` and a derived class `VideoGame` that inherits from it.

## üéÆ Inheritance Example

In this structure, the `VideoGame` class **inherits** all the members (fields and methods) of the `Game` class.

### üìú Base Class (Parent)

```java
public class Game {
    // Field
    private String title;

    // Constructor
    public Game(String title) {
        this.title = title;
    }

    // Public method
    public void startGame() {
        System.out.println("The game '" + title + "' is starting up...");
    }

    // Public void example method
    public void displayGameType() {
        System.out.println("This is a general type of game.");
    }
}
```

-----

### üïπÔ∏è Derived Class (Child)

The `extends` keyword (common in Java; C\# uses a colon `:`) signifies inheritance.

```java
public class VideoGame extends Game {
    private String platform;

    // Constructor: Calls the base class constructor using 'super()'
    public VideoGame(String title, String platform) {
        super(title); // Calls Game(title)
        this.platform = platform;
    }

    // A new method specific to VideoGame
    public void loadLevel() {
        System.out.println("Loading level 1 on " + platform + ".");
    }

    // Overriding the public void example method from the base class
    @Override // Optional annotation, good practice
    public void displayGameType() {
        System.out.println("This is a specific type of game: a Video Game played on " + platform + ".");
    }
}
```

-----

### üíª Example Usage

Here is how you would use these classes, demonstrating how the derived class gets methods from the base class:

```java
public class Main {
    public static void main(String[] args) {
        // Create an instance of the derived class
        VideoGame myGame = new VideoGame("Cyberpunk 2077", "PC");

        System.out.println("**Calling inherited and overridden methods:**");
        
        // 1. Calling the inherited public void method from the Game class
        myGame.startGame(); 

        // 2. Calling the overridden public void example method
        myGame.displayGameType(); 

        // 3. Calling a method specific to the VideoGame class
        myGame.loadLevel();
    }
}
```

### üí° Key Takeaways

1.  **Inheritance:** `VideoGame` automatically has the `startGame()` method from `Game`.
2.  **`public void` Method:** Both `startGame()` and `displayGameType()` are examples of `public void` methods‚Äîthey are **publicly accessible** and perform an action without **returning a value** (`void`).
3.  **Method Overriding:** The `VideoGame` class provides its own implementation of `displayGameType()`, which is called when you use a `VideoGame` object.
